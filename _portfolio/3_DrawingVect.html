--- title: "Line Drawing Vectorization" excerpt: "Paper implementation of Fidelity vs. Simplicity: A Global Approach to Line Drawing Vectorization by Favreau et al.
" collection: portfolio --- This project implements part of the algorithm from Fidelity vs. Simplicity: A Global Approach to Line Drawing Vectorization (Favreau et al., 2016). The goal was to automatically convert raster line drawings into vectorized curve networks that balance accuracy with simplicity, producing a vectorized image.

Key Features

  • Developed a region-based skeletonization algorithm, more robust to noise and sketchy drawings than using simple thinning.
  • Constructed a topological graph from skeletonized drawings using node detection (endpoints, junctions, sharp turns) and edge tracing.
  • Applied adaptive BÃ©zier curve fitting with recursive splitting to achieve high fidelity to input drawings.

Results

  • Achieved high-fidelity curve networks from raster input.
  • Region-based skeletonization outperformed thinning on noisy/sketchy drawings.
  • Outputs tended to create extra short curves, suggesting hypergraph optimization could simplify results.
